# nihitdev.github.io

Nihit's personal GitHub Pages portfolio.

## Publish it

1. Create a **public** GitHub repository named exactly `nihitdev.github.io`.
2. Put these files in the repository root.
3. Push to `main`.
4. In GitHub: **Settings → Pages → Build and deployment → Deploy from a branch → main / (root)**.
5. Wait for deployment, then visit `https://nihitdev.github.io`.

## Customize before publishing

- In `index.html`, replace `hello@example.com` with a real contact address, or remove the email row.
- Add more project cards as you ship projects.
- The portfolio highlights `Oxide Terminal` and `yoo`. Aether-Log is deliberately not listed.

## Local preview

Open `index.html` in a browser. No build tools required.

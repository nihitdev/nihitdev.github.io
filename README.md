# Nihit Sunhare Portfolio

A premium, animated developer portfolio built with React, Vite, Tailwind CSS, Motion, and Lucide icons.

## Stack

- React
- Vite
- Tailwind CSS
- Motion
- Lucide React

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy to `nihitdev.github.io`

1. Copy this project into the root of the `nihitdev.github.io` repository.
2. Push it to the `main` branch.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **GitHub Actions**.
5. The included workflow builds and deploys the site automatically.

## Custom domain

After `nihit.is-a.dev` is approved and active:

1. Open **Settings → Pages** in the repository.
2. Set the custom domain to `nihit.is-a.dev`.
3. Enable **Enforce HTTPS** after GitHub issues the certificate.

## Personalize

Edit `src/data/portfolio.js` to change skills, projects, links, and text.
